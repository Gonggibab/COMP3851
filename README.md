## 환경 설정 및 실행 방법

1. 레포 클론:

```sh
git clone https://github.com/Gonggibab/COMP3851.git
```

2. NPM package 설치:

```sh
npm install
```

3. 개발모드로 코드 실행:

```sh
npm start
```
